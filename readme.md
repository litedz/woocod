# WooCoD DZ Landing Page

Official landing page for WooCoD DZ, a WooCommerce Cash on Delivery solution designed specifically for Algerian online stores.

## About WooCoD DZ

WooCoD DZ helps WooCommerce merchants simplify Cash on Delivery operations in Algeria by providing:

- Cash on Delivery checkout form
- Wilaya management
- Delivery fee configuration
- Checkout customization
- Order management (Pro)
- Reports and analytics (Pro)

## Local Development

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

Open:

```text
index.html
```

in your browser.

## Deployment

This project is designed for GitHub Pages.

### Enable GitHub Pages

1. Go to Repository Settings
2. Open Pages
3. Select:
   - Source → Deploy from a branch
   - Branch → main
   - Folder → /root
4. Save

Your website will be available at:

```text
https://YOUR_USERNAME.github.io/YOUR_REPOSITORY/
```

## Custom Domain

If you own a domain:

```text
woocod-dz.com
```

Add it in:

```text
Repository Settings → Pages → Custom Domain
```

Then configure your DNS records.
